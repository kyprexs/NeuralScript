Getting Started with NeuralScript

Learn how to install NeuralScript and write your first program

Installation

NeuralScript is currently in alpha development. Follow these steps to get started:

Prerequisites

  • Python 3.9 or higher
  • LLVM 15+ (for JIT compilation)
  • CUDA Toolkit (optional, for GPU acceleration)
  • Git

Clone and Build

git clone https://github.com/kyprexs/NeuralScript.git
cd NeuralScript
pip install -r requirements.txt
python setup.py build

Verify Installation

python -c "from compiler.lexer import tokenize; print('NeuralScript ready!')"

üß† Neural Networks

Built-in support for machine learning:

use ml::*

struct SimpleNet {
    layer1: Dense<784, 128>,
    layer2: Dense<128, 10>,
}

impl SimpleNet {
    fn forward(&self, x: Tensor) -> Tensor {
        let h1 = self.layer1.forward(x).relu()
        self.layer2.forward(h1).softmax()
    }
    
    // Automatic backward pass generation
    #[autodiff(forward)]
    fn backward() -> Self::Gradient { }
}
